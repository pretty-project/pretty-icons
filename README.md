
# pretty-icons

### Overview

The <strong>pretty-icons</strong> is a simple Material Icons / Material Symbols
icon set implementation for Clojure projects.

### deps.edn

```

                                     :sha     "xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx"}}}
```
